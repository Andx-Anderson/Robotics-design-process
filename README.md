# Robotics Design Process Website

An interactive educational website that explains the engineering design process in robotics through clear visuals, structured steps, and real-world examples.

---

## 🔗 Live Demo
[Click Me](https://andx-anderson.github.io/Robotics-design-process/)

---

##  Overview
This project is a front-end website designed to help students understand how engineers design, test, and improve robots.

Instead of just reading definitions, users can explore the engineering design process through interactive elements, visual layouts, and real-world examples like self-driving cars.

---

##  Features
- Interactive engineering design process cycle
- Step-by-step breakdown of robotics design stages
- Real-world example (self-driving car)
- Clear comparison of criteria vs constraints
- Failure → improvement visualization
- Glossary of key engineering terms
- Smooth animations and scroll reveal effects
- Fully responsive design (mobile-friendly)

---

##  Tech Stack
- HTML5
- CSS3 (custom styling + animations)
- JavaScript (DOM interaction + UI logic)
